# Notepad++ Macro Creator

Convert a pre-written text/code to macro (XML).

## shortcuts.xml

```xml
<NotepadPlus>
    <InternalCommands>
		<!-- Shortcut -->
    </InternalCommands>
    <Macros>
        <!-- Insert Macro Here -->
    </Macros>
    <UserDefinedCommands>
        <!-- Command -->
    </UserDefinedCommands>
    <PluginCommands />
    <ScintillaKeys />
</NotepadPlus>
```
